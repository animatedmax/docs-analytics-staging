Pivotal Product Documentation : Configuring the vSphere Environment

This page last changed on Nov 22, 2013 by rawlie.

The steps in this section configure your VMware vSphere environment for installation of Pivotal AX.

  1. Login to the vCenter installation using VMware vSphere client software.
    This software is available only for the Microsoft Windows platform.
  2. Select the Inventory > Hosts and Clusters view.
  3. Click Create a datacenter.
    A new datacenter icon displays in the navigation tree.
  4. Enter a name for the datacenter.
  5. Select the new datacenter in the navigation tree.
  6. Click Create a cluster. 
    The New Cluster Wizard opens and displays the Cluster Features screen.
  7. Enter a name for the cluster.
  8. Under Cluster Features, select Turn on vSphereDRS and click Next.
    The vSphere DRS configuration screen displays.
  9. Select Fully Automated or Partially Automated (Consult your vSphere documentation for more information on which option to choose.)  and click Next.
    The Power Management screen displays.
  10. Select Off and click Next.
    The Virtual Machine Swapfile Location screen displays.
  11. Select Store the swapfile in the same directory as the virtual machine and click Next.
    The New Cluster Wizard displays the selected options and allows you to correct them if necessary. 
  12. Click Finish.
  13. Select the cluster you created.
  14. Click Add a Host.
    The Add Host Wizard displays.
  15. Enter the host name or IP address of the ESXi server for this host.
  16. Enter the username for the administrative account for this host. 
    A Security Alert dialog box may appear. Click Yes.
    The Assign License screen displays.
  17. Select the appropriate license for your vSphere deployment and click Next.
    The wizard displays the host configuration for review.
  18. Click Next.
    The Configure Lockdown Mode screen displays.
  19. Do not select any options and click Next.
    The Choose Resource Pool screen opens.
  20. Select the default option to put all of the host's resources in the cluster's root resource pool and click Next.
  21. Review the host options you have selected and click Finish.
  22. Select the Inventory > VMs and Templates view.
  23. Select the data center you created in an earlier step.
  24. Click the create folder icon to create a folder under the data center. Name the folder vm_folder.
  25. Click the create folder icon to create a folder under the data center. Name the folder templates_folder.
  26. Select the Inventory > Datastores and Datastore Clusters view.
  27. Select your datacenter.
    Within the datacenter folder you should see a datastore whose name has been auto-generated. If the name contains spaces, rename the datastore so that there are no spaces in the name.
  28. Select the datastore.
  29. Click Browse this datastore.
    A file browser window displays.

You have now configured your vSphere environment. Continue the Pivotal AX installation by deplyong the Pivotal CF Installer virtual machine.
