VMware vSphere/ESXi Requirements

  • VMware vSphere version 5.0, 5.1, or 5.5 deployment. (version 5.1 is recommended)
  • VMware vSphere edition: Standard and above
  • VMware vSphere access should include user privileges to create virtual machines, disk paths, and virtual machine folders. 
  • vSphere client software (runs on Microsoft Windows only) 

For more information on ESXi hardware requirements, see the VMware vSphere documentation. 

Networking Requirements

  • Ability to create a wildcard DNS record to point to your router or load balancer

Pivotal AX Software Requirements

  • Pivotal HD version 1.0.1, 1.0.3 or 1.1
  • One of the following Web browsers:
    • Microsoft Windows Internet Explorer 9 or later
    • Firefox: most-recent version and the preceding two versions
    • Google Chrome: most-recent version and the preceding two versions
  • SMTP server that is network-accessible from the Pivotal AX installation
