Deploying the Pivotal CF Operations Manager Virtual Machine

In this section, you deploy a VMware OVF template to create the Pivotal CF Operations Manager virtual machine. This virtual machine runs the Pivotal CF Operations Manger application that you will use to provision the virtual machines for Pivotal AX and to install Pivotal AX software. 

You perform these steps from within the VMware vSphere Web client that is connected to your vSphere environment.  

  1. Using the VMware vSphere Web client, log in to your vSphere environment.
  2. Select the Hosts and Clusters view.
  3. Right-click on the data center and select Deploy OVF template.
  4. Select Local File and click the Browse button to open a file browser.
  5. Browse to the ova template you received with your Pivotal AX software and click Next.
  6. Verify the OVF template details and click Next.
  7. Accept the license agreement and click Next.
    The Select Name and Folder screen displays.  
  8. Enter a name for the virtual machine, such as Pivotal CF.
  9. Select the data center from the list. 
  10. Click Next.
    The Disk Format screen displays and shows the name of the datastore you created.
  11. Select the datastore.
  12. Select Thick Provision Lazy Zero and click Next.
    The Network Mapping screen displays.  
  13. Select a destination network and click Next.
    If you have defined a vlan or other networking,  select that networking destination. Otherwise, select the default, VM Network. 
    The Properties screen displays. 
  14. Enter the IP Address, Netmask,  Default Gateway, and DNS fields with the appropriate entries for your networking environment.
  15. Enter and confirm the Admin Password for this virtual machine and click Next.
    The username for this VM is tempest. Use this username and the Admin password for SSH access to the Pivotal CF Operations Manager virtual machine. 
    The Ready to Complete screen displays.

  16. Select Power on after deployment and click Finish.
    vCenter deploys and starts the virtual machine. This may take up to 15 minutes to complete.

  17. Continue with the installation procedure using the Pivotal CF Operations Manager Web application.

    See Configuring Pivotal CF Operations Manager to Install Pivotal AX.
