Pivotal Product Documentation : Deploying the Pivotal CF Installation Virtual Machine

This page last changed on Nov 22, 2013 by rawlie.

Before starting the steps in this section, complete the steps in Configuring the vSphere Environment.

In this section, you deploy a VMware OVF template to create the Pivotal CF virtual machine. This virtual machine runs the Pivotal CF Installer that you will use to provision the virtual machines for Pivotal AX and to install Pivotal AX software. 

  1. Select the Inventory > Hosts and Clusters view.
  2. Select (what?? datacenter? templates folder?  (jack is checking w/ tempest team)  (datacenter, I think) I removed the previous step to create templates folder, 
  3. Click File > Deploy OVF template.
  4. Click the Browse button to open a file browser.
  5. Browse to the ova template you received with your Pivotal AX software and click Next.
  6. Verify the OVF template details and click Next.
  7. Accept the license agreement and click Next.
    The Name and Location screen displays.  
  8.  Select your datacenter. 
  9. Enter a name for the virtual machine, such as Pivotal CF.
    The Host/Cluster screen displays.
  10. Select the cluster you created.
    The Specific Host screen displays.  
  11. Select the ESXi host in this cluster. 
    The Disk Format screen displays and shows the name of the datastore you created.
  12. Select Thick Provision Lazy Zero and click Next.
    The Network Mapping screen displays.  
  13. Select a destination network and click Next. If you have defined a vlan or other networking,  select that networking destination. Otherwise, select the default, VM Network. 
    The Properties screen displays. 
  14. Enter the IP Address, Netmask,  Default Gateway, and DNS fields with the appropriate entries for your networking environment.
  15. Enter and confirm the Admin Password for this virtual machine and click Next.
    The username for this VM is tempest ??really?. Use this username and the Admin password for SSH access to the Pivotal CF Installer virtual machine. 
    The Ready to Complete screen displays.

  16. Select Power on after deployment and click Finish.
    vCenter begins to deploy and start the virtual machine. This may take up to 15 minutes to complete.

  17. Continue with the installation procedure using the Pivotal CF Installer.

    See Configuring the Pivotal CF Installer to Install Pivotal AX.
