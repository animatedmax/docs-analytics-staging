System Architecture

Logging in to Pivotal AX

After the installation of Pivotal AX is complete you use a Web browser to access the Pivotal AX application.  

From this application, you can:

  • Create projects
  • Define data sources for analysis
  • Visualize data
  • Manage your Pivotal AX installation

To login to Pivotal AX:

  1. Enter the URL of your installation in your Web browser
    The Pivotal AX login screen displays:
  2. If this is the first time you are logging in to Pivotal AX, click the New User? Register link.
    The new user form displays.  
     
  3. Fill in the form and click Submit.
    An email with a confirmation link is sent to the specified email address.
  4. Click the confirmation link.
    The Pivotal AX login screen displays. 
  5. Enter your new user name and password.
    The Pivotal AX application opens in your Web browser. 

Global Settings

To configure global settings:

  1. Click the user name drop-down list in the upper-right corner and select Global Settings .
      

  2. Configure the following settings that apply to all projects in your Pivotal AX application:

    SettingDescription
    Data Sampling

    Data sampling allows you to limit the number of events ingested by Pivotal AX for analysis. When you set this sampling interval, only events on the interval are processed. For example, set the Sample every drop-down list to 10 events to process only every 10th record of your data. You can also choose intervals of 100, 1000, 5000, or 10000 events.

    You can also override the global setting for Data Sampling for each project. See Changing Project Settings.

    Data SearchSets the period of time used for searching data in Canvas View or Auto Charts view. Only events within the specified time period are included in the search results. You can chose One Day, One Week, One Month, One Year, or All.
    Status Notifications

    Sets the type of notifications sent from Pivotal AX. Choose one or both of the following:

    • By Email—Sends email notifications to the email address defined in the profile.
    • On Browser —Displays notifications in the Web browser where Pivotal AX is running.

    Note: To enable notification on Firefox browser, enter 'about:config" in the browser's address bar, and set "network.websocket.allowInsecureFromHTTPS" to true. Save your changes.

  3. Click Save Changes.

Managing Projects

A project is a grouping of your data sources and data visualizations of that data. You can perform the the following actions to manage your projects:

  • Add a new project
  • Change project settings
  • Share a project

Add a new project

To add a new project: 

  1. Select All Projects in the left menu bar and click New Project.
  2. Enter a name and description for your project.
  3. Click Add Project.
    Your new project is created.

For information about adding data sources, see Data Source Management Guide.

Changing Project Settings

To override the global setting for Data Sampling for a project:

  1. Select a project by selecting the All Projects in the left menu bar and clicking on a project name. 
  2. Click the Manage icon in the left menu bar.
  3. Click Project Settings.
     
  4. Select Override Global Settings.
  5. Enable or disable sampling by checking or unchecking Enable sampling.
  6. Set the number of events to sample using the Sample every drop-down list.
  7. Click Save Changes.

Sharing Projects

You can share a project with other Pivotal AX users.

To share a project:

  1. Select a project by selecting All Projects icon in the left menu bar and clicking on a project name. 
  2. Click the Manage icon in the left menu bar and select Project Sharing. 
  3. Click Share This Project.
  4. Enter the email addresses of the users with whom you want to share your project. Separate the addresses with commas.
  5. (Optional) Enter the text of a message to include with the email that invites the users to share your project.
    The users receive an email inviting them to share your project and a list of project members displays.
  6. Click Send Invite.
    Pivotal AX sends an email inviting the users to share the project.   

Deleting Projects

To delete a project:

  1. Select Projects > View All Projects
    A box displays containing information about each project defined in your account.
  2. Click the Trash icon in the box containing the project you want to delete. 
  3. Confirm the deletion.
    The project is deleted.  


User Management

You can perform the following actions to manage Pivotal AX users:

  • Set up a new user
  • Modify your user profile
  • Change your password
  • View account overview
  • Viewing projects

Setting up a new user

To set up a new user account:

  1. In a Web browser, navigate to Pivotal AX.
  2. Click the New User? Register link. 
  3. Enter the following user information:
    • The name of the business (optional)
    • Email address of user
    • Password
    • First and Last Name 
    • Phone number
    • City, State, and country
  4. Click the box to indicate that you agree with the User Agreement.
  5. Click Submit.
    The user receives an email with instructions for setting up their account.

Managing your user profile

To manage your user profile:

  1. Login to Pivotal AX.
  2. Click the user name drop-down list in the upper-right corner and select Profile.
  3. Change any of the following fields:
    • First and Last Name 
    • Business name
    • Phone number
    • City, state, and Country
  4. Click Save Changes.

Changing your password

To change your password:

  1. Login to Pivotal AX.
  2. Click the user name drop-down list in the upper-right corner and select Password. 
  3. Enter your current password.
  4. Enter the new password, and confirm. 
  5. Click Save Changes.


View your Account Overview

You can view information about your projects such as the total amount of data processed, top projects, the number of events processed, and usage history.  

To view the Account Overview for your projects:

  1. Login to Pivotal AX.
  2. Click the user name drop-down list in the upper-right corner and select  Account Overview .
